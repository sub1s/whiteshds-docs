###### 通过缩短IP地址（通过[0xInfection](https://twitter.com/0xInfection)）绕过WAF

可以通过删除零来缩短IP地址：

```
http://1.0.0.1 → http://1.1
http://127.0.0.1 → http://127.1
http://192.168.0.1 → http://192.168.1

http://0xC0A80001 or http://3232235521 → 192.168.0.1
http://192.168.257 → 192.168.1.1
http://192.168.516 → 192.168.2.4
```

> 这会绕过用于SSRF，开放重定向等的WAF过滤器，在这些过滤器中，任何IP输入都被列入黑名单。

有关更多信息，请参见[如何隐藏任何URL](http://www.pc-help.org/obscure.htm)和[Magic IP地址的快捷方式](https://stuff-things.net/2014/09/25/magic-ip-address-shortcuts/)。
